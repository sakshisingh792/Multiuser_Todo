# 📝 Multi-User Todo Application (Django)

A full-stack Todo application built using **Django** with complete user authentication and authorization.

This project allows multiple users to manage their own personal todo lists securely.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup / Logout)
- 🛡 Authorization (Users can only see their own todos)
- ➕ Add new todo
- ✏ Edit existing todo
- ✅ Mark as completed / pending
- 🗑 Delete todo
- 🎨 Responsive UI using Bootstrap


---

## 🛠 Tech Stack

- **Backend:** Django
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite (default Django DB)
- **Authentication:** Django built-in auth system
